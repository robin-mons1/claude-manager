# Claude Phone Hub

A web-based terminal manager for Claude Code CLI with mobile access support.

## Features

- Manage multiple Claude Code terminals from a single interface
- Password-protected access
- Real-time sync between desktop and mobile
- Swipe navigation on mobile devices
- Remote access via ngrok

## Quick Start

```bash
# Install dependencies
npm install

# Set up your password
npm run setup

# Start development server
npm run dev
```

Open http://localhost:5173 in your browser.

## Remote Access (Phone)

1. Install ngrok: https://ngrok.com/download

2. authenticate grok

3. run dev program, this automatically starts up ngrok.
## Configuration

Edit `.env` to configure:
- `PORT` - Server port (default: 3000)
- `PASSWORD_HASH` - Generated by setup script
- `JWT_SECRET` - Generated by setup script

## Development

- Frontend: `npm run dev:client` (Vite, port 5173)
- Backend: `npm run dev:server` (Express, port 3000)
- Both: `npm run dev`

## Production Build

```bash
npm run build
npm run start
```
